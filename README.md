# candyapi
API build for Practice of PassPort.js
- Angular Project
- Heroku
## Local Setup
> Step 1:  

**Download zip file**  
or  
**Clone** : `$ git clone https://github.com/chiragbudhiraja/candyapi.git`


>Step 2:  

#### To install dependencies  
`$ npm install `  
#### To Run locally  
`$ npm start `
